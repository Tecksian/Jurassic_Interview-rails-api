    Create a dinosaur with no species key¶ ↑

    POSTing to /dinosaurs with:
{
 "dinosaur" : {
     "name" : "Testasaurus-1",
     "species_name": "Triceratops"
 }
}

    returns:

    {
  "response": {
    "name": "Testasaurus-1",
    "species_name": "Triceratops",
    "is_carnivore": false,
    "cage_id": null
  }
}

    Create a dinosaur with a species key¶ ↑

    {
 "dinosaur" : {
     "name" : "Simone2",
     "species":
     {
         "name" : "Velociraptor"
     }
 }
}

    returns: {

    "response": {
  "name": "Simone2",
  "species_name": "Velociraptor",
  "is_carnivore": true,
  "cage_id": null
}

    }

    Try to add another dinosaur with the same name¶ ↑

    POST /dinosaurs with:
{
 "dinosaur" : {
     "name" : "Simone2",
     "species_name" : "Stegalosaurus"
 }
}

    returns:

    {
  "error": "conflict",
  "error_description": "The request conflicts with an existing resource.",
  "name_already_taken": "Simone2"
}

    {
      "name": "Simone2",
      "species_name": "Velociraptor",

    }
  ],
  "count": 2
}

    View only dinosaurs that are triceratops¶ ↑

    GET /dinosaurs?species_name=Triceratops
{
  "response": [
    {
      "name": "Triceratops",
      "is_carnivore": false
    }
  ],
  "count": 1
}

    Create a cage¶ ↑

    POST /cages
{
     "cage":
     {
         "max_occupancy" : 3,
         "status" : "ACTIVE"
     }

}

    returns

    {
  "response": {
    "id": 15,
    "max_occupancy": 3,
    "status": "ACTIVE",
    "current_occupancy": 0
  }
}

    Power off a cage¶ ↑

    POST /cages
{
     "cage":
     {
         "id": 15,
         "status" : "INACTIVE"
     }

}

    returns nothing, as expected.

    Put a dinosaur in a cage¶ ↑

    PUT /dinosaurs/to_cage with:
    {
     "dinosaur" : {
         "name" : "Simone1",
         "cage":
         {
             "id" : "15"
         }
     }
    }

    returns:

    {
  "response": {
    "name": "Simone0",
    "species_name": "Triceratops",
    "is_carnivore": false,
    "cage_id": 15
  }
}

    Try to power off a cage with a dino in it:¶ ↑

    /cages/15 with:
{
     "cage":
     {
         "id": 15,
         "status" : "INACTIVE"
     }

}

    returns:

    {
  "error": "invalid_resource",
  "error_description": "The current resource was deemed invalid.",
  "messages": {
    "powered_up": [
      "Cannot have a dinosaur in a cage that is powered off. "
    ]
  }
}
